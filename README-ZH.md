# Paste Newbee

[English](README.md) | 中文

本仓库包含Paste Newbee的网站 - 一款支持Windows、macOS和iOS平台的智能剪贴板同步工具。

## 关于Paste Newbee

Paste Newbee是一款智能剪贴板工具，可在多设备间实现实时同步。它支持多种内容格式，包括文本、富文本、图片和文件。

主要特性：

- 跨设备实时同步
- 支持多种格式（文本、图片、文件）
- 智能分类和搜索
- 隐私保护和安全
- 现代UI设计，支持浅色/深色主题
